O que é HTML?

Linguagem de marcação usada para criar páginas web

O que é CSS?

Folhas de estilo usadas para definir o design de páginas web

O que é JavaScript?

Linguagem de programação usada para adicionar interatividade em páginas web

O que é uma variável?

Um contêiner para armazenar dados que podem mudar durante a execução do programa

O que é um loop?

Uma estrutura que repete um bloco de código até uma condição ser satisfeita

O que é uma função?

Um bloco de código reutilizável que executa uma tarefa específica

Desenvolvido para estudo interativo de programação!
